## Android Studio로 나만의 미니 포토샵 어플 만들기
### 포토샵 기능 
     1. 갤러리에서 사진 가져오기
     2. 줌 인, 줌 아웃
     3. 회전
     4. 명암 조절
     5. 색 반전 후 명암 조절
     6. 흑백 변환
     7. 블러링 필터 적용
     8. 눈 내리기 효과
     9. 모든 효과 지우기
     10. 펜 그리기
     11. 스탬프 찍기
     12. 모자이크 영역 선택
     13. 지우개
     14. 편집한 이미지 갤러리에 저장

### 프로젝트 한계점
+ 펜 그리기, 스탬프 찍기 기능을 일부 편집 기능 (사진 크기 조절, 색 반전 등)과 중첩 사용하기 힘들다.
     + 편집 기능을 먼저 적용하여 사진 저장 후에 펜 그리기, 스탬프 찍기를 적용하도록 한다.
